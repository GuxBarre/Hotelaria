Descrição detalhada

Classe responsável por armazenar e validar um número de telefone. Formato esperado: +XXYYZZZZZZZZZ (Ex: +5511988887777) - Total 14 caracteres. Nota: O código valida apenas dígitos após o '+', ignorando formatação visual como '(', ')' ou '-'.
