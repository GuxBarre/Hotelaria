\doxysubsection{Descrição detalhada}
Representa o domínio de senha para autenticação de gerentes. 


\begin{DoxyItemize}
\item 
\end{DoxyItemize}

Regras de formato válido conforme especificação\+:
\begin{DoxyItemize}
\item {\bfseries{Tamanho}}\+: Exatamente 5 caracteres.
\item {\bfseries{Caracteres permitidos}}\+: Letras (A-\/Z, a-\/z), dígitos (0-\/9) e caracteres especiais (! " \# \$ \% \& ?).
\item {\bfseries{Regras de Sequência}}\+:
\item Letra não pode ser seguida por letra.
\item Dígito não pode ser seguido por dígito.
\item {\bfseries{Regras de Composição}}\+: Deve conter pelo menos uma letra maiúscula, uma letra minúscula, um dígito e um caractere especial. 
\end{DoxyItemize}
