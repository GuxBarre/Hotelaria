\doxysubsection{Descrição detalhada}
Classe responsável por representar e validar um número de cartão de crédito. 


\begin{DoxyItemize}
\item Esta classe armazena um número de cartão e garante que ele atenda aos requisitos\+:
\item Ter exatamente 16 dígitos numéricos.
\item Ser válido segundo o algoritmo de Luhn (verificação de dígito verificador). 
\end{DoxyItemize}
