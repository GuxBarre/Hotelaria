Classe para representação e validação de datas. 


\begin{DoxyItemize}
\item Armazena internamente a data em inteiros (dia, mês, ano), mas permite a manipulação via string (geralmente no formato D\+D/\+M\+M/\+A\+A\+AA ou similar). 
\end{DoxyItemize}
