Representa o domínio de Código de identificação. 


\begin{DoxyItemize}
\item 
\end{DoxyItemize}

Regras de formato válido conforme especificação\+:
\begin{DoxyItemize}
\item Deve conter exatamente 10 caracteres\mbox{[}cite\+: 53\mbox{]}.
\item Cada caractere deve ser uma letra (a-\/z) ou um dígito (0-\/9)\mbox{[}cite\+: 54\mbox{]}.
\item Este identificador é utilizado como chave primária para Hotéis e Reservas. 
\end{DoxyItemize}
