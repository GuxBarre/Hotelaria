\doxysubsection{Descrição detalhada}
Representa o domínio de capacidade de acomodação de um quarto. 


\begin{DoxyItemize}
\item 
\end{DoxyItemize}

Esta classe armazena e valida a capacidade de hóspedes. De acordo com as regras de negócio, o formato válido aceita apenas\+:
\begin{DoxyItemize}
\item 1, 2, 3 ou 4 pessoas.
\item Qualquer valor fora deste conjunto é considerado inválido. 
\end{DoxyItemize}
